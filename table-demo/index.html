Table DEMO

A table is a way of creating a structured grid display of data the paired tags (open?close) we will be interested in are:

  • <table> : identifies the start of the table
  • <thead> : identifies the header row of the table
  • <tbody> : identifies the data detail body of the table
  • <tfoot> : identifies the footer row of the table
  • <tr> : identifies the start of a row on the table
  • <th> : identifies the cell in the header row for the coloum title
  • <td> : identifies the cell on the detail row or footer row to hold content

some tag have parameters. The border parameter of the table applies a border around your table. the colospan parameter and rowspan parameter are used on the <td> tab allow you to merge colums on a single row or column cells across rows.
